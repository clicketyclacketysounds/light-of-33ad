# The Light of 33 AD

Sunlight that reflected off Earth in 33 AD has been traveling outward ever since.
It is now an expanding shell of light almost 2,000 light-years wide. This project
explores where that light is now, and how much of a living Earth it still carries.

**Live site:** https://clicketyclacketysounds.github.io/light-of-33ad/

The home page is a menu linking to four ways into the story:

- **The Animation** ([animation.html](animation.html)) an interactive, self-contained
  Canvas piece that rides the shell outward past the stars it has already passed.
- **The Treatment** ([treatment.html](treatment.html)) a long-form, magazine-style
  web essay.
- **The White Paper** ([whitepaper.html](whitepaper.html)) the formal physics and
  information-theory analysis, with every claim labeled fact, assumption, or uncertain.
- **The Slides** ([The-Light-of-33-AD.pptx](The-Light-of-33-AD.pptx)) a nine-slide,
  plain-language tour of the key findings.

Every page is a single self-contained HTML file, no build step and no server, so
it runs anywhere GitHub Pages can serve a static file.
